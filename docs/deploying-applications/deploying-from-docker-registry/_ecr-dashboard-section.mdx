In order to link an existing ECR registry from the dashboard, navigate to the **Integrations** tab and select **Docker Registry**. Then, select **Add a Registry** in top right, and select **Elastic Container Registry**. Give the registry a name, for example `my-awesome-registry`:

![ECR](https://files.readme.io/a9b7bee-Screen_Shot_2021-12-31_at_7.53.06_AM.png)

Next, input the AWS credentials (if you do not have your AWS credentials, follow the [instructions below](#creating-an-aws-ecr-user)). Then, click **Save Settings**.
