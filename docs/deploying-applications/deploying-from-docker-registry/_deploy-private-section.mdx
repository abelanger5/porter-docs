To deploy from a private image, make sure you have [linked a private registry](./linking-existing-registry). Then, simply select the image repo and tag you would like to use:

![Image repo selection](https://files.readme.io/9d796f4-Screen_Shot_2021-03-18_at_11.26.45_AM.png "Screen Shot 2021-03-18 at 11.26.45 AM.png")

Select "Continue" after specifying your image.
